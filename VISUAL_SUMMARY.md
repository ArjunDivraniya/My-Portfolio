## 🎬 Animation Features

### 1️⃣ Glowing Path
```
┌─────────────────────┐
│                     │
│  ╔═════════════════╗│
│  ║  ═══════════╍  ║│  ← Draws as you scroll
│  ║     Yellow    ║│
│  ║     Glow      ║│
│  ║     Effect    ║│
│  ║    ═══════╍   ║│
│  ║                ║│
│  ╚═════════════════╝│
│                     │
└─────────────────────┘
```

### 2️⃣ 3D Card Reveals
```
Start → Enters Viewport → Animates
  ↓           ↓              ↓
Hidden    Y-Rotation      Visible
Flying    + Scale up      + Glowing
Left/Right (0.8→1.0)       Border
(-100/+100) (45°→0°)
```

### 3️⃣ Pulsing Nodes
```
Viewport Event          Animation
     ↓                      ↓
   ENTER          ┌─────────────────┐
     ↓            │  Start Pulse    │
   Pulse On  →    │  Scale 1→1.3→1  │
     ↓            │  Repeat -1      │
   LEAVE          │  Glow Intensify │
     ↓            └─────────────────┘
   Pulse Off
```

### 4️⃣ Hover Effects
```
Card Hover → Lift up + Glow ↑ + Border Animation →
```

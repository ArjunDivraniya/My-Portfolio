## 🎬 Animation Features

### 1. Glowing Path
- SVG line draws downward
- Yellow gradient glow
- Scroll-linked animation

### 2. 3D Card Reveals
- Alternate left/right entrance
- Y-axis rotation (3D flip)
- Scale animation (0.8 → 1.0)

### 3. Pulsing Nodes
- Pulse on viewport entry
- Yellow glow intensifies
- Auto-stops on viewport exit

### 4. Hover Effects
- Cards lift up
- Border glow increases
- Top border animates

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| No animation | Ensure GSAP registered |
| Glow not visible | Check filter support |
| Cards misaligned | Clear cache, check viewport |
| Mobile lag | Increase scrub value |
| SVG line missing | Verify SVG namespace |
| Colors wrong | Check Tailwind config |

## 💡 Pro Tips

1. **Test on Real Devices**
   - Emulation isn't always accurate
   - Use actual smartphones/tablets

2. **Monitor Performance**
   - Chrome DevTools → Performance tab
   - Look for dropped frames

3. **Mobile First**
   - Design for mobile
   - Scale up for desktop

4. **Accessibility**
   - Test with reduced motion
   - Check contrast ratios

5. **Performance Counts**
   - Every millisecond matters
   - Lazy load where possible
